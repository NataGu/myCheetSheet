HTML Tables

Table Tags

Tag Name Description
<table> Table Content that is used to represent a two-dimensional table made of rows and columns.
<thead> Table Head Headings of table columns encapsulated in table rows.
<tbody> Table Body The set of rows containing actual table data.
<tr> Table Row Used to add rows to a table before adding table data and table headings.
<td> Table Data To add a cell of data to a table.
<tfoot> Table Foot To give footer content or to summarize content at the end of a table.

Table Attributes

Attribute Name Description
colspan Column Span Defines how many columns a td element should span.
rowspan Row Span Defines how many rows a td element should span.
